## How to add a new painting

1. Take a photo of it (natural light, straight-on, no flash)
2. Drop the file into `Massimo-Artwork/paintings/` (or `Massimo-Artwork/digital/` for digital work)
3. Tell Claude: *"Add [painting name] to the gallery — it's oil on canvas, 2025"*
4. Claude handles the rest
